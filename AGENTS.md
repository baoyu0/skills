## Loading skills

To load a skill, reference its path:
- `skill_view(name='methodology/code-assembly')`
- `skill_view(name='agent-orchestration/hermes-do-task')`
- `skill_view(name='content-publishing/obsidian-halo')`
- `skill_view(name='media/subtitle-edit')`

Each skill directory contains a `SKILL.md` with full instructions.
